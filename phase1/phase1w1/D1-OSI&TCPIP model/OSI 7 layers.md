# OSI 七層模型

OSI 模型用來描述資料傳輸的概覽，與主流的 TCP/IP 相容。

## 七層架構（由上至下）

| 層號 | 名稱 | 英文 | 功能說明 |
|------|------|------|---------|
| L7 | 應用層 | Application | 使用者互動，提供網路服務給應用程式 |
| L6 | 呈現層 | Presentation | 資料格式轉換、加解密（SSL/TLS） |
| L5 | 會話層 | Session | 控制與管理通訊協定，連結 L4 與 L6 |
| L4 | 傳輸層 | Transport | 實際傳輸內容，端對端可靠性 |
| L3 | 網路層 | Network | Routing、IP 定址、封包傳遞 |
| L2 | 資料連結層 | Data Link | 乙太網路、MAC 位址、frame 處理 |
| L1 | 實體層 | Physical | 實際硬體、線路、電氣訊號 |

## 重要工具

- **Wireshark**：可以抓取並分析各層的封包內容
