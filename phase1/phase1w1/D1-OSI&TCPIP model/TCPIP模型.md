# TCP/IP 模型

現代網路主流模型，將 OSI 七層簡化為四層。

## 四層架構

### 網路存取層（L1 + L2）
- 負責處理資料如何通過實體網路傳輸
- 涵蓋硬體、MAC 位址、frame 處理

### 網路層（L3）
- 負責 packet 封裝與傳輸路由選擇（Routing）
- **主要協定**：IP

### 傳輸層（L4）
- 負責端對端（end-to-end）的資料傳輸
- 確保傳遞可靠、錯誤處理、流量控制
- **主要協定**：TCP（可靠）、UDP（不可靠）

### 應用層（L5 + L6 + L7）
- 應用程式之間的實際通訊
- 資料呈現、格式轉換、加解密
- **主要協定**：HTTP、HTTPS、FTP、DNS…

## TCP/IP vs OSI 對應

| TCP/IP 層 | OSI 層 |
|-----------|--------|
| 應用層 | L5 會話、L6 呈現、L7 應用 |
| 傳輸層 | L4 傳輸 |
| 網路層 | L3 網路 |
| 網路存取層 | L1 實體、L2 資料連結 |
