# CLI 三欄速查表

| 功能          | CMD                                  | PowerShell             | Linux                  |
| ----------- | ------------------------------------ | ---------------------- | ---------------------- |
| 查網卡資訊       | `ipconfig /all`                      | Get-NetIPAddress       | `ifconfig` / `ip addr` |
| 查連線狀態       | `netstat -ano`                       | Get-NetTCPConnection   | `netstat -an`          |
| 查路由表        | `route print`                        | Get-NetRoute           | `route -n`             |
| 查 ARP table | `arp -a`                             | Get-NetNeighbor        | `arp -a`               |
| 查防火牆狀態      | `netsh advfirewall show allprofiles` | Get-NetFirewallProfile | `ufw status`           |
| 追蹤路由路徑      | `tracert`                            |                        | `traceroute`           |
| DNS 查詢      | `nslookup`                           |                        | `nslookup` / `dig`     |
| 查執行中程序      | `tasklist`                           | Get-Process            | `ps aux`               |
| 查系統資訊       | `systeminfo`                         | Get-ComputerInfo       | `uname -a`             |
| 查開放 Port    | `netstat -ano`                       |                        | `ss -tlnp`             |

# Port速查

| Port  | 協議      | 服務         | 功能                    |
| ----- | ------- | ---------- | --------------------- |
| 22    | TCP     | SSH        | 加密遠端連線，取代 Telnet      |
| 23    | TCP     | Telnet     | 不加密遠端連線，已淘汰           |
| 25    | TCP     | SMTP       | 寄送電子郵件                |
| 53    | UDP/TCP | DNS        | 網域名稱解析（IP ↔ 網址）       |
| 80    | TCP     | HTTP       | 未加密網頁傳輸               |
| 443   | TCP     | HTTPS      | 加密網頁傳輸（TLS）           |
| 3389  | TCP     | RDP        | Windows 遠端桌面控制        |
| 3306  | TCP     | MySQL      | MySQL 資料庫連線           |
| 5432  | TCP     | PostgreSQL | PostgreSQL 資料庫連線      |
| 6379  | TCP     | Redis      | Redis 快取資料庫連線         |
| 27017 | TCP     | MongoDB    | MongoDB 資料庫連線         |
| 8080  | TCP     | HTTP alt   | HTTP 替代 port，常用於開發測試  |
| 8443  | TCP     | HTTPS alt  | HTTPS 替代 port，常用於開發測試 |
