# D2 補充說明

## 1. RFC 1918 Private IP 位址範圍

原筆記只提到 Private IP 的概念，但沒有列出實際範圍。這三段是保留給私有網路使用的 IP：

| 範圍 | CIDR | 可用位址數 | 對應 Class |
|------|------|-----------|-----------|
| 10.0.0.0 – 10.255.255.255 | 10.0.0.0/8 | 約 1,677 萬 | Class A |
| 172.16.0.0 – 172.31.255.255 | 172.16.0.0/12 | 約 104 萬 | Class B |
| 192.168.0.0 – 192.168.255.255 | 192.168.0.0/16 | 約 6.5 萬 | Class C |

> 家用路由器最常見的就是 `192.168.1.x` 或 `192.168.0.x`，屬於 Class C 私有範圍。

## 2. 其他特殊用途位址

| 位址 / 範圍 | 用途 |
|------------|------|
| 127.0.0.0/8 | Loopback，`127.0.0.1` 最常用 |
| 169.254.0.0/16 | APIPA（Link-Local），無 DHCP 時自動指派 |
| 0.0.0.0 | 代表「未指定」，DHCP 流程中 Host 還沒拿到 IP 時使用 |
| 255.255.255.255 | Limited Broadcast，送給同網段內所有裝置 |

## 3. IPv6 簡介

IPv4 的 32-bit 位址空間（約 43 億個）已耗盡，IPv6 使用 128-bit 解決這個問題。

| 項目 | IPv4 | IPv6 |
|------|------|------|
| 位元數 | 32 bit | 128 bit |
| 表示方式 | 點分十進位（192.168.1.1） | 冒號十六進位（2001:0db8::1） |
| 位址數量 | 約 43 億 | 約 3.4 × 10³⁸ |
| Loopback | 127.0.0.1 | ::1 |
| 私有位址 | RFC 1918 | fc00::/7（ULA） |
| NAT 需求 | 幾乎必要 | 通常不需要 |

> IPv6 目前與 IPv4 並行（雙棧 Dual Stack），完全取代仍需要時間。
