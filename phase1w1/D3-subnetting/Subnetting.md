# 子網路計算（Subnetting）

## VLSM（Variable Length Subnet Mask）

自訂子網路遮罩，切出剛好符合需求的 Host 數量，避免浪費 IP 位址。

## 計算步驟

1. 決定需要的 Host 數量，選擇最小能容納的 CIDR
2. 用 CIDR 推算可切幾個子網路，找出目標 IP 落在哪一個子網路
3. 找出該子網路的 Subnet Address 與 Broadcast Address

## 範例

**題目**：`10.1.1.224/22`，求 Subnet Address、Broadcast、可用 Host 範圍

```
/22 表示前 22 bits 是 Network 部分
22 - 16 = 6（第三個 Octet 借了 6 bits 給 Network）
子網路間距 = 2^(8-6) = 4，子網路為 0, 4, 8, 12...（第三個 Octet）

10.1.1.x → 1 落在第三個 Octet 的 0~3 之間

Subnet Address  : 10.1.0.0
Broadcast       : 10.1.3.255
Host Range      : 10.1.0.1 ~ 10.1.3.254
```
